PROMPT LAB

Steam-Style Store Writeup

Prompt Lab is a focused prompt engineering workspace for people who actually iterate, compare, test, and ship prompts.

It is built for real prompt work, not one-off demos. Write raw prompts, enhance them, compare versions, run A/B tests, manage reusable prompt libraries, track evaluation runs, and move from rough draft to production-ready prompt systems without losing the thread.

Whether you are tuning system prompts, building reusable templates, evaluating model behavior, or organizing a serious prompt library, Prompt Lab gives you a single workspace to do the work cleanly.

Core Pitch

Prompt Lab turns prompt iteration into a real workflow:
- draft
- enhance
- compare
- test
- save
- version
- reuse

Instead of juggling raw text files, ad hoc notes, and half-reproducible experiments, you get a dedicated environment for structured prompt development across extension, desktop, and web surfaces.

Why Prompt Lab

- Build prompts with speed and structure
- Save reusable prompts into a searchable library
- Fill template variables before execution
- Compare raw and improved outputs side by side
- Run A/B tests between prompt variants
- Track evaluation runs over time
- Organize prompts by collections and tags
- Keep version history and restore prior prompt states
- Move prompts between library, composer, scratchpad, and testing workflows

Key Features

1. Prompt Creation Workspace
- Write and edit prompts in a dedicated creator surface
- Switch between focused and split layouts
- View improved output, diffs, variants, and notes in one workflow
- Use markdown preview where useful

2. Prompt Enhancement Flow
- Send prompts through supported model providers
- Stream responses on supported direct-fetch paths
- See skeleton loading and live output preview during generation
- Cancel stale or unwanted requests cleanly
- Surface save actions optimistically during active enhancement

3. Prompt Library
- Save prompts with title, tags, collections, and change notes
- Search and filter the library quickly
- Import preset packs into the library
- Detect duplicate or near-duplicate presets before import
- Preserve structured prompt inputs including select-style variables

4. Template Variable Support
- Detect template variables inside prompts
- Open a fill step before applying templated prompts
- Support editable auto-filled ghost variables
- Render select inputs when preset metadata defines enumerated choices

5. A/B Testing
- Send prompts directly from the library into Variant A or Variant B
- Resolve template variables before A/B insertion
- Compare prompt variants in a dedicated experiment flow

6. Evaluation Timeline
- Review run history in a timeline panel
- Filter runs by model, status, and date range
- Compare outputs with diff highlighting
- View delta-style comparison metrics
- Copy comparison results for external review

7. Versioning and Recovery
- Keep version history for saved prompts
- Restore older versions when experimentation goes sideways
- Use change notes to understand what shifted between revisions

8. Multi-Surface Workflow
- Designed to work across:
  - Chrome extension
  - Tauri desktop app
  - web mode
- Keeps prompt work portable across contexts

9. Supporting Workspaces
- Scratchpad-style writing surface
- Composer workflow for assembling blocks
- Test case support for prompt-specific evaluation flows

10. UX Quality Improvements
- Debounced library search
- Toast feedback with motion polish
- Modal and tab transitions
- Batch progress visibility for long-running test operations
- More useful compare and history flows for repeated iteration
